## Instalación

Antes de descargar el repositorio es necesario tener instalados npm, python y uvicorn. Tras esto y una vez clonado el respositorio dirijase al directorio "metaverse-fornt" y haga "npm i" para instalar las dependencias necesarias. 

## Uso 

Para levantar "metaverse" dirijase al directorio correspondiente y escriba en un terminal el comando "uvicorn main:app --reload --host 0.0.0.0 --port 8000"
Para levantar "metaverse-front" dirijase al directorio correspondiente y escriba en un terminal el comando "ng serve"

Una vez levantadas los dos servidores dirijase a un navegador y navegue a la dirección "localhost:4200"
